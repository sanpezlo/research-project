---
Write a JavaScript function that returns in an array the odd numbers from 1 to N using a "while" loop.
---

```js
function oddNumbers(n) {
  let result = [];
  let i = 1;
  while (i <= n) {
    if (i % 2 !== 0) {
      result.push(i);
    }
    i++;
  }
  return result;
}
```
