---
Write a JavaScript function that returns the average of an array of numbers using a "while" loop.
---

```js
function average(numbers) {
  if (numbers.length == 0) {
    return null;
  }

  let sum = 0;
  let i = 0;
  while (i < numbers.length) {
    sum += numbers[i];
    i++;
  }
  return sum / numbers.length;
}
```
