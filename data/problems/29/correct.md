---
Write a JavaScript function that returns the sum of all integers between 1 and N, divisible by 9, using a "while" loop.
---

```js
function countAndSumDivisibleBy9(n) {
  let i = 1,
    sum = 0;
  while (i <= n) {
    if (i % 9 == 0) {
      sum += i;
    }
    i++;
  }
  return sum;
}
```
