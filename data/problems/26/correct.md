---
Write a JavaScript function that returns in an array the first N numbers of the sequence: a(N) = a^N + a (N-1) + N using a "while" loop.
---

```js
function sequence(a, n) {
  let result = [];
  let i = 1;
  while (i <= n) {
    let value = a ** i + a * (i - 1) + i;
    result.push(value);
    i++;
  }
  return result;
}
```
