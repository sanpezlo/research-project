---
Write a JavaScript function that returns the sum of the numbers from 1 to N using a "while" loop.
---

```initial
    let sum = 1;
```

```initial
    let sum = n;
```

```js
function sum(n) {
  let i = 1;
  while (i <= n) {
    sum += i;
    i++;
  }
  return sum;
}
```

```js

  let i = 0;

    i++;
  }
  return sum;
}
```
