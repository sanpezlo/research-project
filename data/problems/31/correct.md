---
Write a JavaScript function that given a string, returns a string with each letter capitalized using a "while" loop.
---

```js
function capitalizeLetters(str) {
  let result = "",
    i = 0;
  while (i < str.length) {
    result += str[i].toUpperCase();
    i++;
  }
  return result;
}
```
