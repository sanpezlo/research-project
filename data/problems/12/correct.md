---
Write a JavaScript function that returns the power of a given base raised to a given exponent using a "while" loop.
---

```js
function power(base, n) {
  let result = 1;
  let i = 0;
  while (i < n) {
    result *= base;
    i++;
  }
  return result;
}
```
