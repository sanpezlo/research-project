---
Write a JavaScript function that returns in an array the even numbers from 1 to N using a while loop.
---

```js
function evenNumbers(n) {
  let result = [];
  let i = 2;
  while (i <= n) {
    result.push(i);
    i += 2;
  }
  return result;
}
```
