---
Write a JavaScript function that returns in an array all factors of a number using a "while" loop.
---

```js
function factorsOfNumber(num) {
  let factors = [];
  let i = 1;
  while (i <= num) {
    if (num % i == 0) {
      factors.push(i);
    }
    i++;
  }
  return factors;
}
```
