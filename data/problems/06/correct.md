---
Write a JavaScript function that returns the average of an array of n using a "while" loop.
---

```js
function min(numbers) {
  let min = numbers[0];
  let i = 0;
  while (i < numbers.length) {
    if (numbers[i] < min) {
      min = numbers[i];
    }
    i++;
  }
  return min;
}
```
